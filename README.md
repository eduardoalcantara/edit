# Editor Linux

**Autor:** Perplexity AI  
**Data:** 2026-06-06  
**Versão:** 1.0

Editor de terminal Linux com foco em usabilidade, previsibilidade e proteção contra perda de trabalho.

## Estrutura

- `specs/` — especificações (`to-do/`, `done/`, `report/`).
- `docs/` — documentação do projeto.
- `scripts/` — automações e utilitários.
- `tests/` — testes automatizados e manuais.
- `src/` — código-fonte principal.

## Documentos principais

- `.cursorrules` — regras de atuação do Cursor.
- `PROJECT_RULES.md` — padrões e decisões permanentes do projeto.
- `PROJECT_STATUS.md` — estado atual e pendências.
- `PROJECT_TIMELINE.md` — histórico de mudanças.

## Objetivo do produto

Criar um editor terminal amigável, com menus, atalhos previsíveis, temas, histórico de arquivos recentes, clipboard interno, modais de confirmação e terminal inferior integrado.
